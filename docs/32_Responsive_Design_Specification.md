# Responsive Design Specification

## Breakpoints
- mobile: 320px+
- tablet: 768px+
- desktop: 1024px+
- large desktop: 1280px+

## Layout rules
- Single-column by default on mobile
- Two-column only when readable
- Universe map should collapse gracefully
- Cards should stack vertically
- Navigation should convert to a mobile drawer

## Typography rules
- Large hero text scales down smoothly
- Body line length should remain comfortable
- No fixed-size text that breaks small screens

## Touch targets
- Minimum comfortable tap size
- Adequate spacing between controls

## Media behavior
- Images should never overflow their containers
- Sample chapter reader should remain distraction-free
- Forms should remain usable with mobile keyboards open
