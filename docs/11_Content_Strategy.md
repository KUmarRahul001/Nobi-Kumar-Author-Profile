# Content Strategy

## Content goals
- Make the author memorable
- Help readers choose a starting point
- Support SEO for titles, author name, and genre searches
- Create recurring reasons to return

## Content pillars
### 1. Books
Synopsis, excerpts, reading order, character notes, universe links

### 2. Author story
Bio, inspiration, writing process, philosophy, milestones

### 3. Universe lore
NNU / Verma Saga map, timelines, connections, motifs

### 4. Reader education
Themes, symbolism, behind-the-scenes notes, FAQs

### 5. Updates
Releases, events, news, announcements

## Tone and voice
- Suspenseful but controlled
- Confident, concise, literary
- Emotionally charged without melodrama
- Clear enough for broad reader comprehension

## Content formats
- Long-form blog posts
- Book cards
- Chapter samples
- Universe annotations
- FAQ entries
- Newsletter issues

## Content operations
- Each book page must have canonical metadata
- Blog content should use a standardized frontmatter schema
- Reusable CTA blocks should be centrally managed
- Sample excerpts should be editable without code changes

## SEO content priorities
- Author name
- Book titles
- Psychological thriller
- Verma Saga
- NNU
- Sample chapter
- Buy now
- Reading order
