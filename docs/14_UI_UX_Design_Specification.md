# UI/UX Design Specification

## UX objectives
- Help visitors understand the author in under 10 seconds
- Help readers reach a sample chapter in under 2 clicks
- Help buyers reach a retailer in under 3 clicks
- Help fans understand universe connections without confusion

## Key UX patterns
### Home hero
- Big title
- One-line positioning
- Featured book CTA
- Secondary CTA to universe map

### Book cards
- Cover
- Title
- Series label
- One-line hook
- Sample / Buy actions

### Universe map
- Zoomable or scrollable visual
- Nodes and relationships
- Clear legend
- Reading-order emphasis

### Blog cards
- Category
- Title
- Read time
- Featured image
- Comment count

### Contact form
- Category selector
- Simple field grouping
- Confirmation state

## Interaction rules
- No noisy carousels
- No autoplay audio
- No random motion
- Forms should validate inline
- Search should update quickly and clearly

## Mobile UX
- Sticky primary CTA only where necessary
- Single-column content flow
- Large tap targets
- Minimal header height

## Information density rule
The visual tone is cinematic, but the user experience must remain easy to scan.
