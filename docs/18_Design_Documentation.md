# Design Documentation

## Visual rationale
The site should look like a thriller film universe because the author brand itself is suspense-driven. Design must reinforce the genre before the visitor reads the first paragraph.

## Content hierarchy rationale
Readers first need:
1. Who is this author?
2. What books are available?
3. Where should I start?
4. How do I buy or sample?

## Conversion rationale
The site should minimize distraction:
- primary CTA: read sample / buy book
- secondary CTA: explore universe
- tertiary CTA: subscribe / contact

## Trust-building elements
- Official biography
- Book catalog completeness
- Sample chapter access
- External retailer links
- Contact and newsletter transparency

## Imagery rationale
Use moody story-world imagery instead of generic author portraits wherever possible, while preserving one clean professional author photo for credibility.

## Editorial design rules
- Shorter paragraphs
- Strong subheads
- Pull quotes for tension
- No wall-of-text blocks
- Use tables only for metadata or comparisons
