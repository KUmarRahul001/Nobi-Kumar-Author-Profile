# Information Architecture

## Top-level navigation
- Home
- Books
- Universe Map
- Blog
- About
- Contact
- Search

## Recommended footer navigation
- Newsletter
- FAQ
- Privacy Policy
- Terms
- Accessibility
- Social links
- Retailer links

## Page hierarchy
### Home
- Hero
- Featured book
- Latest release
- Universe teaser
- Blog teaser
- Newsletter CTA
- Footer

### Books
- All books
- Series groups
- Individual book pages
- Sample chapter pages
- Reading order view

### Universe Map
- Main map
- Series nodes
- Character / location nodes
- Reading-path filters

### Blog
- Blog index
- Category pages
- Article pages
- Comment thread

### About
- Bio
- Writing philosophy
- Publishing journey
- Brand statement

### Contact
- Contact form
- Collaboration route
- Media route

## Navigation rules
- Book pages must be reachable in one click from home
- Search should be visible in the header
- Universe map should be one click from the primary nav
- Newsletter CTA should appear in the header, footer, and blog template
