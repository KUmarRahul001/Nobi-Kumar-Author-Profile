# Wireframe Descriptions

## Home page
Top hero, featured book, latest release, universe teaser, blog teaser, newsletter CTA, footer.

## Books index
Filter bar, grid of books, reading order summary, search field, series sections.

## Book detail
Cover, synopsis, key facts, sample excerpt, retailer links, related books, FAQ, share tools.

## Universe map
Intro text, legend, interactive graph, reading-order cues, side panel for selected node details.

## Blog index
Featured post, categories, chronological list, newsletter CTA.

## Blog article
Title, metadata, article body, pull quotes, related posts, comment section, share actions.

## About
Portrait, bio, writing story, mission, achievements, media quote blocks.

## Contact
Inquiry type selector, form, response expectations, alternate contact routes.

## Search results
Query summary, grouped results, zero-state guidance.

## Mobile wireframe behavior
- header collapses
- hero stacks vertically
- cards become single column
- map simplifies to stacked tiles or canvas with touch controls
