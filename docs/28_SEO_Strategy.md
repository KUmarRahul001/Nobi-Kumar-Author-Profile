# SEO Strategy

## Objectives
- Rank for author name
- Rank for each book title
- Rank for psychological thriller and related genre queries
- Rank for universe / series queries
- Rank for sample chapter searches

## Technical SEO
- Unique titles and meta descriptions
- Canonical tags
- Open Graph and Twitter cards
- XML sitemap
- robots.txt
- Schema.org markup for author, book, blog post, breadcrumb

## Content SEO
- One page per book
- One page per major blog post
- Universe map indexable with explanatory text
- Internal links between books, posts, and about page

## Keyword clusters
- Nobi Kumar
- psychological thriller author
- Verma Saga
- NNU
- sample chapter
- buy book
- novel list
- book order

## Search intent mapping
- Informational: about, universe, blog
- Navigational: book titles
- Transactional: buy book
- Commercial investigation: sample chapter, reading order, reviews

## Rich result opportunities
- Book schema
- Breadcrumb schema
- Article schema
- Person schema
