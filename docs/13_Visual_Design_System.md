# Visual Design System

## Design direction
A dark premium thriller interface with cinematic spacing, controlled contrast, and minimal clutter.

## Layout philosophy
- One strong focal point per section
- Large margins and breathing room
- Deep hierarchy
- Reading-first content flow

## Color system
- Base: black / charcoal / deep grey
- Surface: layered near-black panels
- Accent 1: crimson for danger / tension
- Accent 2: neon violet for mystery / tech / intrigue
- Neutral text: off-white and muted greys

## Surface treatment
- Soft borders
- Subtle glow only on highlighted CTA elements
- Light grain or texture only in hero media areas

## Motion principles
- Slow, elegant, suspenseful
- Avoid playful transitions
- Use purposeful reveal animations

## UI state language
- Hover: slight lift, glow, or underline
- Active: compressed and crisp
- Focus: highly visible accessible outline
- Disabled: reduced opacity, no ambiguity

## Page rhythm
- Hero
- Social proof / value
- Book cards
- Universe block
- CTA
- Supporting content

## Accessibility note
Every dark surface must preserve contrast ratio and remain readable at night and low brightness.
