# Environment Variables

## Application
- NEXT_PUBLIC_SITE_URL
- NEXT_PUBLIC_SITE_NAME
- NEXT_PUBLIC_DEFAULT_THEME
- NEXT_PUBLIC_ANALYTICS_ID

## Database
- DATABASE_URL
- DIRECT_DATABASE_URL

## Email / newsletter
- NEWSLETTER_PROVIDER_API_KEY
- NEWSLETTER_LIST_ID
- NEWSLETTER_FROM_EMAIL

## Contact / forms
- CONTACT_TO_EMAIL
- CONTACT_FROM_EMAIL
- CAPTCHA_SECRET

## Media / storage
- STORAGE_BUCKET
- STORAGE_ACCESS_KEY
- STORAGE_SECRET_KEY

## Security
- SESSION_SECRET
- ADMIN_SECRET
- WEBHOOK_SECRET

## Operational rule
Secrets must only exist in server-side environment configuration.
