# Architecture Decision Records (ADRs)

## ADR-001: Next.js App Router
Decision: Use Next.js App Router for content, SEO, and routing flexibility.
Reason: Strong fit for static and interactive author content.

## ADR-002: Outbound purchase links
Decision: Do not implement checkout in v1.
Reason: Lower scope and aligns with current publishing workflows.

## ADR-003: MDX content
Decision: Use MDX or structured markdown for books and blog content.
Reason: Editable content with strong developer ergonomics.

## ADR-004: Postgres persistence
Decision: Use Postgres for comments, newsletter, and contact records.
Reason: Reliable, familiar, and scalable.

## ADR-005: Dark theme default
Decision: Use dark as the default visual mode.
Reason: Core brand expression and genre fit.
