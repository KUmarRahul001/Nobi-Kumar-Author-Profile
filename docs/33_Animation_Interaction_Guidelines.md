# Animation & Interaction Guidelines

## Motion style
- Slow
- Intentional
- Cinematic
- Suspenseful
- Subtle

## Allowed interactions
- Fade, slight slide, gentle zoom
- Soft card lift
- Progress reveal in sample reading
- Map node focus transitions
- Theme toggle transition

## Disallowed interactions
- Bouncy UI
- Excessive parallax
- Overly playful microinteractions
- Autoplay media

## UX timing
- Keep interactions short enough to feel responsive
- Keep reveals slow enough to feel premium

## Reduced motion
- Honor prefers-reduced-motion
- Provide non-animated equivalents for all critical interactions
