# User Journey Maps

## Journey 1: Discover → Sample → Buy
1. Lands on homepage
2. Sees featured book and thriller positioning
3. Opens book detail page
4. Reads sample chapter
5. Clicks outbound retailer link
6. Leaves with a strong purchase intent

### Success signals
- sample open
- retailer click
- reduced bounce

## Journey 2: Explore universe → understand order
1. Lands on book catalog
2. Opens universe map
3. Reviews chronology and connections
4. Chooses starting point
5. Opens related book detail page

### Success signals
- map engagement
- multi-page session
- reading-order clarity

## Journey 3: Read blog → subscribe
1. Finds blog post through SEO or social
2. Reads article
3. Sees newsletter CTA
4. Submits email
5. Receives confirmation

### Success signals
- long dwell time
- newsletter conversion
- return visit potential

## Journey 4: Contact author
1. Visits about or contact page
2. Chooses inquiry type
3. Sends message
4. Receives confirmation
5. Admin triages response

### Success signals
- form completion
- low spam rate
- clear categorization
